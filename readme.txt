This plugin generate QR code of posts' links in admin section.

== Description ==
QR Coder print a QR code of the posts' permalinks in admin section. You can choose which post types have the QR codes.

== Installation ==

1. Upload `qr-coder` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
